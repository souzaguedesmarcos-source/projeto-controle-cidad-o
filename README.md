# Sistema de Controle - Programa Leite das Crianças

## Integrante
- Marcos Vinicios de Souza Guedes
  
## Problema
Órgãos públicos e escolas precisam administrar o estoque de leite distribuído pelo programa **Leite das Crianças** no Paraná. Atualmente, o controle é pouco organizado e dificulta acompanhar cidades, instituições e beneficiários.

## Objetivo
Criar um sistema web simples para:
- Controlar estoque de leite por cidade e instituição.
- Cadastrar e gerenciar beneficiários.
- Registrar entregas e consumo.
- Emitir relatórios para órgãos públicos.

## Público-alvo
- Escolas e órgãos públicos responsáveis pela distribuição.
- Famílias de baixa renda cadastradas no programa.

## Funcionalidades
- Cadastro de usuários (beneficiários).
- Cadastro de cidades e instituições.
- Controle de estoque (entrada e saída).
- Registro de entregas.
- Relatórios por cidade, instituição e usuário.

## Tecnologias
- Java para back-end.
- HTML, CSS e JavaScript para front-end.
- Banco de dados Local Storage.
- GitHub para versionamento e colaboração.

## Modelagem
Entidades principais:
- Usuário
- Cidade
- Instituição
- Estoque
- Entrega

## Como executar
1. Clonar o repositório.
2. Configurar banco de dados.
3. Executar aplicação Java.
4. Acessar via navegador.
